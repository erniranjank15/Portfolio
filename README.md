
# Niranjan Kasote — Personal Portfolio

A responsive personal portfolio website built with pure HTML, CSS, and JavaScript.

## Live Demo

[personalportfolionk.netlify.app](https://personalportfolionk.netlify.app/)

## Features

- Sidebar layout with profile, contact info, and social links
- Sections: About, Skills, Education, Portfolio
- Responsive design for mobile and desktop
- Smooth hover effects and transitions
- Project cards with live links
